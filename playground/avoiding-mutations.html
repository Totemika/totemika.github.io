Avoiding mutations

Adding a new item to an array
arr.push(newItem)
[
  ...arr,
  newItem
]
Removing an item from an array
arr.splice(index, 1)
[
  ...arr.slice(0, index),
  ...arr.slice(index+1)
]
Incrementing an array item
arr[i]++
[
  ...arr.slice(0,i),
  arr[i]+1,
  ...arr.slice(i+1)
]
Modifying an object property
obj.prop = !obj.prop
Object.assign({}, obj, {
  prop: !obj.prop
})
OR
{
  ...obj,
  prop: !obj.prop
};
